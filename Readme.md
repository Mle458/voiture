# POC

This poc is an API to list cars and add comments on cars. You must login to see and add comments.

# CONFIGURATION

This POC use MYSQL database. To launch this poc, you must do some configurations of database

- database url
- database username
- database password

in application.properties file

# TEST

A default was added with the username 'defaultuser' and password 'password'.
Some cars are also added.

A swagger can be consulted at /v2/api-docs and /swagger-ui.html#/
